# Book_Hub

카카오 검색 API 를 이용해 책 검색 사이트 

### 기본환경 ###
포트는 8000 을 사용합니다. 
http://localhost:8000/

### 서비스 스펙 ###
1. Java 8 버전을 사용
2. framework - spring-boot (Maven)
3. server - Tomcat 사용 
4. Database - h2 (maven dependency)
5. css, js - bootstrap

### 추가 모듈(라이브러리) ###
1. httpclient - for RestAPI connected
2. jquery-3.2.1 - Front-end framework.
3. bootpag - jQuery plugin

### 구동 ###
1. spring-boot 실행
2. Import > 
3. Check out Maven Projects form SCM > 
SCM URL : https://github.com/Hi-Hub/Book_Hub.git >
Finish
